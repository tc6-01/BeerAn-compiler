# BeerAn编译器

## content：
flex & bison & llvm & c++14

## usage:
```shell
    git clone https://github.com/tc6-01/BeerAn-compiler.git
    cd BeerAn-compiler
    make
    ./BeerAn < BeerAn.bc > BeerAn.txt
    cat BeerAn.txt
```
